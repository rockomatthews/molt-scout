## What are OpenClaw Skills?

OpenClaw Skills are custom extension scripts (typically written in Node.js or Python) that expand the capabilities of the OpenClaw AI agent. The core OpenClaw engine provides the autonomous framework, and installing **Skills** enables the agent to complete specific external tasksâ€”such as scheduling actions on **Moltbook**, moderating **Telegram** groups, generating SEO keyword ideas, analyzing cryptocurrency signals, or integrating with third-party APIs (e.g., Twitter or Reddit).

## How to Install Custom Skills in OpenClaw

Follow these simple steps to get started with any skill.

1

### Download the Skill File

Browse the directory and download the .json or .js skill file (or the latest release package) you want to use.

2

### Locate Skills Folder

Open your local OpenClaw installation directory and find the /skills (or /plugins) folder. If your setup differs, follow your OpenClaw config/docs.

3

### Import the Script

Move the file into the skills/plugins folder. If you run OpenClaw in Docker, make sure the folder is volume-mounted so changes persist.

4

### Restart & Verify

Restart OpenClaw and verify the skill loads correctly. Use the built-in skill listing command (e.g., /skills) or check logs for errors.
