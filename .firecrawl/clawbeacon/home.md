## Everything You Need to Command Your Agents

Built with modern tech stack for maximum performance and developer experience

### Agent Management

Monitor and manage all your AI agents in one place. See who's working, who's idle, and what everyone's up to.

### Kanban Board

Drag-and-drop task management with real-time updates. Organize work across backlog, in progress, review, and done.

### Live Agent Feed

Real-time stream of agent communications and status updates. Stay informed about everything happening in your system.

### Real-time Sync

Server-sent events (SSE) keep your dashboard perfectly in sync. Changes appear instantly, no refresh needed.

### Mobile First

Fully responsive design that works beautifully on any device. Manage your agents from anywhere.

### Open Source

MIT licensed and fully transparent. Fork it, customize it, make it yours. Contributions welcome!
