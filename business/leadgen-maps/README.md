# LeadGen Maps

A non-trading money path: **sell lead lists**.

## Pitch
"Tell me the niche + city. I’ll deliver 200+ verified local businesses (CSV) in under an hour."

## Inputs
- niche keyword(s)
- city / region
- optional filters: rating >= X, has website, has phone, etc.

## Outputs
- `leads.csv`
- `leads.json`
- short summary (counts by rating, top suburbs, etc.)

## Tooling options
- `skills/google-maps-scraper` (Omkar Cloud) playbook
- (fallback) Google Places API (we already have local-places skill)

## Monetization
- One-off: $49–$199 per list
- Subscription: $199–$999/mo for weekly refresh

## Guardrails
- No spam/outreach automation by default.
- Respect applicable privacy laws.
